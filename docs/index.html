La Gran Transmutación

En el arte antiguo de la alquimia, el Magnum Opus — la Gran Obra — no era meramente la búsqueda del oro, sino la transformación sistemática de la materia prima en algo de valor perdurable mediante la aplicación de conocimiento, disciplina y voluntad.

Alchemical Agent Ecosystem aplica este mismo principio a la inteligencia: los datos crudos entran como prima materia, pasan por una red soberana de agentes IA especializados vinculados en Círculos Alquímicos dinámicamente auto-formados, y emergen como acción coherente, memoria duradera y decisión rastreable.

Todo orquestado completamente en tu propio hardware, con cero dependencia de APIs externas o nubes costosas. El motor de inferencia IA está impulsado por KiloCode AI Gateway — un agregador de LLMs compatible con OpenAI con tier gratuito.

Instalación Rápida

Listo en 5 minutos. Solo necesitas Docker 24+ y Docker Compose v2.

Prerrequisitos

Docker 24+ Docker Compose v2 Git 16 GB RAM KILO_API_KEY (gratis)
1

Clonar el repositorio

bash 
git clone https://github.com/smouj/alchemical-agent-ecosystem.git
cd alchemical-agent-ecosystem
2

Ejecutar el asistente de instalación

bash 
./install.sh --wizard

Este comando configura KiloCode AI, genera el archivo .env y valida el stack automáticamente.

3

Iniciar el ecosistema

bash 
./scripts/alchemical up-fast

Verificar la instalación

bash 
curl -fsS http://localhost/gateway/health | jq .
Después de instalar

Accede al Dashboard

Una vez que el stack está corriendo, abre el Agent Node Studio y observa la magia en tiempo real.

Preguntas Frecuentes

¿Necesito una API key de OpenAI?

No. Alchemical usa KiloCode AI Gateway que tiene un tier gratuito con modelos como Claude, GPT y Gemini. No se requiere tarjeta de crédito.

¿Cuánta potencia de hardware necesito?

Mínimo 16 GB de RAM y 20 GB de disco. Para uso producción recomendado: 32 GB RAM + 50 GB SSD.

¿Puedo ejecutarlo en producción?

Sí. El ecosistema está diseñado para producción con RBAC, API keys, OpenTelemetry, y monitoreo integrado.

¿Qué hace diferente a Alchemical de n8n o LangFlow?

Alchemical es 100% local-first con memoria jerárquica persistente, 10 agentes nomeados especializados, y un motor de orquestación LangGraph. Sin dependencia de clouds externas para funcionar.
